# Boton de panico

La UNACAR preocupada por la seguridad de las universitarias, propone la construcción de una aplicación que ofrezca apoyo para evitar:

* acoso 
* situaciones que atenten contra su seguridad física de las unniversitarias.

y que brinde:

* acompañamiento físico/virtual
* información sobre lugares/senderos seguros

Propuesta: crear una aplicación llamada "botón de pánico"

## App botón de pánico para la UNACAR.

App para apoyo a mujeres en el campus universitario que ofrece acompañamiento físico y/o virtual.

## Publico Objetivo

Dirigida a mujeres en un campus universitario, centrada en la seguridad física de las usuarias, en situaciones de emergencia o en las que se sientan potencialmente inseguras.

## Objetivo

Desarrollar una app que permita a una usuaria compartir su ubicación y solicitar acompañamiento precincial y/o virtual
